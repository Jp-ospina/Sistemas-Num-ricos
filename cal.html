Calculadora de bases de sistemas númericos

En esta sección, se pueden realizar calculos aritmeticos sencillos entre bases numéricas como base 2 (binario), base 8 (octal), base 10 (decimal), base 16 (hexadecimal); se pueden realizar sumas, restas, multiplicaciones, divisiones, potencias y módulos entre dos números.


NOTA:

  • Use '+' para sumar
  • Use '-' para restar
  • Use '*' para multiplicar
  • Use '^' para potencia
  • Use '/' para dividir
  • Use '%' para hallar el módulo
  • Use '**' para radicales (Primero el radicando y luego el indice)
  • Use '//' para logaritmos (Primero el argumento y luego la base)
